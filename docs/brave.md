# Brave Browser Setup Guide

## Extensions

Install all extensions by running the script below, then confirm each install in the browser:

```bash
bash scripts/install-brave-extensions.sh
```

Or install individually via the links in each section.

> **uBlock Origin settings** — export/import via Dashboard → My filters → Backup to file. Store backup in `config/brave/ublock-origin-backup.txt` (gitignored if it contains custom rules with sensitive domains).
> **Proton Pass** - create PIN for additional security.
